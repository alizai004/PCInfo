# PCInfo
A clean, polished, lightweight app that tells you about your system specifications.
Windows Compatibility

Windows 10 / Windows 11 (64-bit) ONLY

This application will not run on:

Windows 7

Windows 8 / 8.1

Python does NOT need to be installed.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 Antivirus / SmartScreen Notice

This app is built using PyInstaller and is signed with a self-signed certificate.

Because of this, some antivirus software (such as Windows SmartScreen or Avast) may incorrectly flag the app as suspicious.

This is a false positive.

------------------------------------------------------------------------------------------------------------------------------------------------------------------------------
 How to Use

Download and extract the ZIP file

Open pcinfo.exe

If prompted by Windows, choose More info → Run anyway
